## Installation

```bash
# clone to repository
$ git clone https://github.com/rafael-camara/ignite-lab-04-nodejs.git

# navigate to folder
$ cd ignite-lab-04-nodejs

# with npm
$ npm install

# with yarn
$ yarn
```

## Running the app with npm

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app with yarn

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
